THANK YOU FOR PURCHASING THE VIRAL POST BOT!

Here is how to set it up in 2 minutes:

STEP 1: INSTALL PYTHON
1. Go to python.org and download the latest version.
2. IMPORTANT: When installing, check the box that says "Add Python to PATH".

STEP 2: INSTALL THE LIBRARIES
1. Open your Command Prompt (Search "cmd" in Windows start bar).
2. Copy and paste this command and hit Enter:
   pip install streamlit openai

STEP 3: RUN THE BOT
1. Open Command Prompt again.
2. Type this command and hit Enter:
   streamlit run viral_bot.py
3. The bot will open in your web browser automatically!
